Introduction to OpenDS Development
As an open source application, OpenDS is available for anyone to use and extend, as well as for individuals to contribute to its ongoing development. This document attempts to provide a set of information that may be helpful to individuals that wish to get involved in the development of OpenDS, those who wish to create plugins and other extensions, or those who wish to merely become more familiar with the server source code layout and components.
