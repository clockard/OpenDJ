Utilities

The idea here is to provide a top-level toolbox for console-specific stuff like preferences and managing administrative users as well as for "support" types of tools that are not part of the directory service per se.

I've included some things which in the end may not make sense, but sort of express the spirit of the sub-panel. These include things like task scheduling which are configured at the server level, but it might be useful in order to manage tasks in a multi-server way -- argument being that since functionally related servers will likely be configured to perform identical tasks, the user might want to schedule those tasks in a sequence (e.g., to limit performance impact schedule tasks to occur in sequence on a set of servers contingent upon success). This is most easily accomplished if the UI supports multiple servers. Likewise, it could be convenient to view and configure logger settings and certificates centrally, and so they've been included in this section as well.
