About This Mockup

This is a static html mockup representing the current design direction for OpenDS Control Center, a browser-based administrative console for managing OpenDS instances. There some "Design Notes" in the various sections of the left navigation panel that provide more information on the planned content. The mockup is designed around the current Administrative Object Model, which is described on the OpenDS wiki.

Several important caveats to note:

  • The design is incomplete and in flux. There are stubs to content that will eventually get filled in (or, alternatively, the stubs might simply disappear)
  • There are things that are broken in this mockup that won't be broken in the implementation. The implementation will be built using the JSF Woodstock components, which will behave properly. This mockup uses the images, css and an approximation of the html generated by the Woodstock components to create a relatively high-fidelity representation of the real deal, but it is limited by its static nature. For example:
    • Things like keyboard navigation do not yet work properly.
    • The browser's back button will not work properly -- this is due to the nature of the javascript used to load links.
    • Likewise, reloading the page will bring you back to this page.
    • For simplicity's sake, the mockup was designed using Firefox as the primary test platform. It'll probably look OK on other browsers, but then again it may not.
