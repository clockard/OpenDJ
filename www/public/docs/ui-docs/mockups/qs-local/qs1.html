The OpenDS QuickSetup tool can be used for basic configuration of a new server instance.

You can also use run QuickSetup via Java Web Start to download, install and configure a new server instance or upgrade an existing server instance to the latest weekly build. To run QuickSetup via Java Web Start click the Launch QuickSetup link on the About QuickSetup Tool page of the OpenDS Wiki.

OpenDS requires a Java SE 5.0 or higher runtime. Additional information on QuickSetup is available on the OpenDS documentation wiki.
