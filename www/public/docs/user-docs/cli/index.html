Command Line Usage Guide
OpenDS includes a number of command-line utilities that may be used to interact with the server. Some of them are administrative tools for managing the server, while others are client applications that may be used to communicate with the server. It also includes a number of other general utilities for performing directory-related tasks.

This page includes a list of the command-line utilities provided with OpenDS and links to information about those tools. Note that for all tools, you can provide a "--help" command-line argument to obtain basic usage information.


Administrative Tools
  • setup -- Perform initial Directory Server configuration.
  • start-ds -- Start the Directory Server.
  • stop-ds -- Stop the Directory Server.
  • import-ldif -- Import LDIF data into the Directory Server.
  • export-ldif -- Export Directory Server data to an LDIF file.
  • backup -- Back up one or more Directory Server backends.
  • restore -- Restore a backup of a Directory Server backend.
  • verify-index -- Perform validation of Directory index data.


LDAP Client Utilities
  • ldapsearch -- Perform LDAP search operations.
  • ldapmodify -- Perform LDAP modify, add, delete, and modify DN operations.
  • ldappasswordmodify -- Perform LDAP password modify operations.
  • ldapcompare -- Perform LDAP compare operations
  • ldapdelete -- Perform LDAP delete operations.


Other OpenDS Tools
  • encode-password -- Encode and compare user passwords.
  • ldifsearch -- Perform search operations in LDIF files.
  • ldifmodify -- Perform modify, add, and delete operations in LDIF files.
  • ldif-diff -- Identify the differences between two LDIF files.
  • list-backends -- Display information about the available backends.
  • make-ldif -- Generate sample LDIF data based on a template file.
