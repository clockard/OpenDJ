The make-ldif Utility

The make-ldif utility may be used to generate LDIF data based on a template file. This can be used to construct any amount of realistic sample data that is suitable for use in applications like performance and scalability testing or attempting to reproduce a problem observed in a production environment.

The complete set of arguments that may be used with the makeldif tool includes:
  • -t {path}
    --templateFile {path}

    This argument specifies the path to the template file describing the data to be generated. This is a required argument.

  • -o {path}
    --ldifFile {path}

    This argument specifies the path to the LDIF file to which the generated data should be written. This is a required argument.

  • -s {seed}
    --randomSeed {seed}

    This argument specifies the integer value that should be used to seed the random number generator. If a random seed is provided, then generating data based on the same template file with the same seed will always generate exactly the same LDIF output. If no seed is provided, then the same template file will likely generate different LDIF output each time it is used.

  • -H
    --help

    This argument will display usage information for the make-ldif utility and will exit without attempting to perform any additional processing.
