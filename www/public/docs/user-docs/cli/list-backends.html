The list-backends Utility

The list-backends utility may be used to obtain information about the backends defined in an OpenDS instance. It has three modes of operation:
  • When invoked with no arguments, it will display the bakcend IDs for all backends configured in the server, along with the base DNs for those backends.
  • When used with the --backendID argument, it will list all of the base DNs for the backend with the specified backend ID.
  • When used with the --baseDN argument, it will list the backend ID of the backend that should be used to hold the entry with the given DN and will also indicate whether that DN is one of the configured base DNs for that backend.

As mentioned above, this utility does not require any command-line arguments. However, the following arguments are available for use:
  • -n
    --backendID

    This argument may be used to specify the backend ID of a backend for which the tool should display the associated base DN(s). This argument may be used multiple times to display the base DNs for multiple backends.

  • -b {dn}
    --baseDN {dn}

    This argument may be used to specify a DN for which the tool should determine which (if any) backend should hold an entry with that DN. It will also indicate whether the specified DN is a base DN for that backend.

  • -H
    --help

    This argument will cause the tool to display its usage information and exit immediately without taking any other action.
