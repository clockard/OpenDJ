Accessing OpenDS configuration through jconsole

jconsole java tool (Java 5 version  - Java 6 version) is a JMX-compliant GUI tool that connects to a running JVM which started with the management agent. This generic tool can be easily used to access and modify the OpenDS configuration information.

How to Start jconsole

Start jconsole by typing  jconsole in a command shell.

Note: To run jconsole from the command line, you must add "JAVA_HOME/bin" to your path, where "JAVA_HOME" is the directory containing the JDK. Alternatively, you can enter the full path when you type the command.

For more information on using jconsole, see Using jconsole (Java 5 version  - Java 6 version)

How to Access the OpenDS Server from jconsole

Using J2SE 5.0

In order to connect jconsole to the OpenDS server, you should use the Advanced tab of the connection window pop up:






JMX URL service:jmx:rmi:///jndi/rmi://<host>:<port>/org.opends.server.protocols.jmx.client-unknown

Where:
  • <host> is a host name, an IPv4 numeric host address, or an IPv6 numeric address enclosed in square brackets.
  • The <port> is the decimal port number of the JMX connector. (See How to Setup an OpenDS server instance.)
Note: The default JMX URL is "service:jmx:rmi:///jndi/rmi://127.0.0.1:1689/org.opends.server.protocols.jmx.client-unknown".
Username A valid LDAP user name.

Note: The default Directory Manager user name is "cn=directory manager".
Password The corresponding LDAP password.

Note: The default password for the Directory Manager is "password".

Using JAVA 6

In order to connect jconsole to the OpenDS server, you should use the "Remote Process" fields:




JMX URL service:jmx:rmi:///jndi/rmi://<host>:<port>/org.opends.server.protocols.jmx.client-unknown

Where:
  • <host> is a host name, an IPv4 numeric host address, or an IPv6 numeric address enclosed in square brackets.
  • The <port> is the decimal port number of the JMX connector. (See How to Setup an OpenDS server instance.)
Note: The default JMX URL is "service:jmx:rmi:///jndi/rmi://127.0.0.1:1689/org.opends.server.protocols.jmx.client-unknown".
Username A valid LDAP user name.

Note: The default Directory Manager user name is "cn=directory manager".
Password The corresponding LDAP password.

Note: The default password for the Directory Manager is "password".

How to View and Modify the OpenDS Server Configuration from jconsole

Once connected to the OpenDS server, the jconsole tool will display management objects (Mbeans). The tree on the left panel shows all the MBeans currently available.
